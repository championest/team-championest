# Team Championest Dashboard

## Deploy
1. ไปที่ https://app.netlify.com/drop
2. Drag & drop โฟลเดอร์ `team-dashboard/` ลงไป
3. ได้ URL ทันที — เปิดบนมือถือได้เลย

## PIN
Default PIN: `7777`
เปลี่ยนได้ใน index.html บรรทัด: `const CORRECT_PIN = "7777";`

## Features
- ดู/แก้ไข team members ทุกคน
- เพิ่ม/ลบ member ได้
- Notes to Claude — เขียนโน้ตไว้ Claude อ่านตอนเริ่ม session
- Real-time sync via Firebase Firestore

## Notes ทางเลือก (Obsidian)
แก้ไขไฟล์ใน Obsidian iOS:
`Second Brain/📝 Notes to Claude.md`
